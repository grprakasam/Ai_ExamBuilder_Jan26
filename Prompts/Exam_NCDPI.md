

# Goal
Create a North Carolina End-of-Grade (EOG) style practice test 


------------------------------------------------
# template for user input

User UI to have the following inputs:
• Grade: [GRADE LEVEL]
• Subject: [SUBJECT]
• Strand / Standard Focus: [SPECIFIC STANDARD OR TOPIC]
• Number of Questions: [NUMBER]
• Question Type: [Multiple Choice / Open-Ended / Mixed]
• Difficulty: On-grade-level, slightly rigorous (EOG-aligned)


------------------------------------------------

# Requirements

Follow these STRICT requirements:

1. Alignment & Rigor
   - Questions must be aligned to NCDPI standards for the specified grade.
   - Match EOG cognitive demand (Recall, Skill/Concept, Strategic Thinking).
   - Avoid trick questions or content above grade level.

2. Question Design
   - Use real EOG-style wording and structure.
   - Avoid unnecessary context unless it supports reasoning.
   - Use grade-appropriate vocabulary.
   - For Math:
     • Include computation, reasoning, and word problems.
     • Avoid calculator dependency unless grade-appropriate.
   - For Science:
     • Emphasize understanding of concepts, cause-effect, and data interpretation.

     
   - For open-ended questions:
     • Provide a brief expected answer or scoring guidance (1–2 lines max).
